The Microsoft Certified: Azure Administrator Associate (AZ-104) is the workhorse certification of the Azure track — and it has a deserved reputation for being harder than its "associate" label suggests. It tests whether you can actually administer an Azure environment: identities, governance, storage, compute, networking, and monitoring, often down to portal-blade and CLI-flag level of detail.

## Who should take it

AZ-104 is aimed at administrators with at least six months of hands-on Azure experience. It's the standard credential for cloud ops, infrastructure, and platform roles in Microsoft shops. Coming straight from [AZ-900](/exams/az-900) without portal time is possible but painful — the exam assumes you've clicked through these workflows before.

## How the exam is scored

Expect roughly 40–60 questions in about 100 minutes, scored on a 1,000-point scale with a 700 pass mark. Microsoft's formats are the most varied of any vendor: multiple choice, drag-and-drop, hotspot dropdowns, and case studies. Some exams include a lab-style section performed in a live Azure environment, so genuine portal fluency matters.

## Where candidates struggle

Networking is the domain that fails most people. Network security group evaluation order, user-defined routes, VNet peering (and what it doesn't do — no transitive routing), Azure Firewall vs NSGs, load balancer SKUs, and private endpoints all appear repeatedly. If you study one domain to excess, make it this one.

Identity questions are subtler than they look: Entra ID licensing tiers, dynamic groups, self-service password reset requirements, and the difference between Azure roles and Entra roles. Storage questions love SAS tokens, access tiers, replication options (LRS/ZRS/GRS/RA-GZRS), and Azure Files identity-based access.

## How to prepare

- **Get a free Azure account and build.** Create a VNet with peered spokes, deploy a VM scale set behind a load balancer, configure an NSG and watch it block traffic, set up a storage account with lifecycle rules. The exam's detail level punishes purely theoretical study.
- **Learn both portal and CLI.** Questions freely mix the portal, Azure CLI, and PowerShell. You don't need to memorise every flag, but you should recognise correct command shapes.
- **Drill the governance hierarchy.** Management groups → subscriptions → resource groups → resources, and how policy, RBAC, and locks inherit down it.
- **Practise structured questions.** Hotspot and drag-and-drop formats reward exam-format familiarity as much as knowledge.

## Common pitfalls

Time management on case studies (read the questions before the scenario), forgetting that VNet peering isn't transitive, and mixing up which replication options protect against regional outage. Also know Azure Backup vs Azure Site Recovery cold — one is backup, one is disaster recovery, and the exam checks you know which solves which requirement.

## After you pass

The certificate is valid for one year and renews free online. From here, admins typically go deeper into networking (AZ-700) or security (AZ-500), or broaden with the [Solutions Architect track](/exams/saa-c03) on AWS if they work multi-cloud.
